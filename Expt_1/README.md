# Experiment 1: Setting up an Oscilloscope

### 1. Objective
* To master the initial power-up and safety sequence of a dual-channel 20MHz oscilloscope.
* To identify and correctly adjust general, vertical, horizontal, and triggering controls.
* To learn how to achieve a sharp, stable trace by adjusting intensity and focus.
* To verify the oscilloscope's basic functionality using its internal calibration (CAL) output.
* To understand the visual representation of voltage (amplitude) and time (period/frequency) on a display.

---

Based on the laboratory manual:
* **Dual channel 20MHz oscilloscope**

### 4. Procedure
According to the manual instructions, the following detailed steps were performed to initialize and test the equipment:

**1. General Controls**:
* Set the **Intensity** control to about three-quarters of its travel.
* Set the **Mode** control to the CH1 (or CHA) position.

**2. Vertical Controls**:
* Set the **Input Coupling** control for both channels to the AC position.
* Set the **Vertical Attenuation** control for both channels to the 1V/div position.
* Set the **Vertical Attenuation Calibration** control for both channels to the detent (locked) position.
* Set the **Vertical Position** control for both channels to about the middle of their travel.

**3. Horizontal Controls**:
* Set the **Horizontal Timebase** control to the 0.5ms/div position.
* Set the **Horizontal Timebase Calibration** control to the detent (locked) position.
* Set the **Horizontal Position** control to about the middle of its travel.

**4. Triggering Controls**:
* Set the **Sweep Mode** control to the AUTO position.
* Set the **Trigger Level** control to the detent (locked) position (or middle of travel).
* Set the **Trigger Source** control to the CH1 (or INT) position.
* Set the **Trigger Source Coupling** control to the AC position.
* Set the **Slope (or Sync)** control to the "+" position.

**5. Powering Up and Testing**:
* Switched on the scope and allowed it to warm up for approximately 30 seconds until a trace appeared.
* Adjusted the **Intensity** and **Focus** controls to ensure the trace was sharp and at an appropriate brightness.
* Connected the Channel 1 input to the scope's **CAL output** using the oscilloscope lead.
* Verified that a stable square wave appeared on the display, confirming the scope was ready for measurement.

### 5. Reflection / Learning Summary
Setting up the oscilloscope correctly is the most important first step in any lab because it ensures all future measurements are accurate. In this experiment, I learned how to "find" and stabilize a signal by adjusting the vertical controls for voltage and the horizontal controls for time. I discovered that the triggering system is essential for freezing a moving wave so it can be studied, and using the internal CAL signal proved that my equipment was working perfectly. By learning how to adjust the focus and intensity, I can now produce clear waveforms that are easy to read. This lab gave me the confidence to use the oscilloscope as a reliable tool for seeing how electricity behaves, which is a skill I will need for every experiment that follows with the Emona trainer.
