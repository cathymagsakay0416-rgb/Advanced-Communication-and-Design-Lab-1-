# Advanced-Communication-System-and-Design-Laboratory (LAB 1)

This repository serves as my official submission for laboratory report 1 for Advanced Communication and Design. It is organized into a specific directory structure to ensure all documentation and data are easily accessible.

## General Laboratory Objectives
Across these four experiments, the primary goals are:
* To master the operation of a **Dual channel 20MHz oscilloscope** for signal visualization.
* To gain proficiency in utilizing the **Emona Telecomms-Trainer 101** modules and faceplate guides.
* To bridge the gap between theoretical communication equations and physical hardware modeling.
* To accurately capture, document, and analyze real-time waveforms using standard lab leads.
* To develop a professional engineering portfolio using Markdown for clear technical reporting.

## Standard Lab Report Template
As per the requirement, every `README.md` inside the experiment folders includes:
1. **Objective**: Specific goals for the individual lab.
2. **Materials and Components Used**: Hardware like the scope and trainer.
3. **Circuit Diagram**: Image links to setup schematics.
4. **Code**: Inline or linked files (if applicable).
5. **Results and Discussion**: Analysis of results and waveforms.
6. **Reflection / Learning Summary**: Summary of key takeaways.
